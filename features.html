Units of Measure

Switch between all popular visual acuity formats instantly — no setup required between patients.

  • US Snellen — 20/200 … 20/20, 20/15, 20/10
  • Metric — 6/60 … 6/6
  • LogMAR — 1.0 through 0.0 (ETDRS standard)
  • Decimal — 0.1 through 1.0

Calibration

Configure AcuityMaster once for your exam lane. Settings persist between sessions.

  • Unit of measure selection (US, Metric, LogMAR, Decimal)
  • Patient-to-screen distance in feet or meters
  • Pediatric fixation video clip selection

Chart Types

Every specialist test in one application — no switching software between chart types.

  • Snellen — standard acuity chart
  • ETDRS — Early Treatment Diabetic Retinopathy Study
  • Worth-Dot-Four — color-correct, calibrated
  • Astigmatic Dial — Jackson cross-cylinder refinement
  • Fixation Disparity — binocular vision testing
  • Contrast Sensitivity — beyond standard Snellen

Optotypes

Alternative optotypes assess visual acuity across all patients — including pre-literate children and non-English speakers.

  • Letters — standard Sloan / Snellen alphabet
  • Pictures — updated images children recognise today
  • Tumbling E — for patients who cannot name letters
  • Landolt C — ISO standard broken ring
  • Numbers — Arabic numerals

Display Mode

Change display mode instantly without leaving the chart — no reconfiguration needed between patients.

  • Single Letter — isolate one optotype to eliminate crowding. Particularly useful for amblyopia testing.
  • Single Line — display one complete acuity line at a time.
  • Multiline — full chart view in traditional Snellen format.

Pediatric Friendly

AcuityMaster is designed with pediatric patients in mind. Updated picture optotypes include objects children actually recognise today — including a cell phone, because today’s kids no longer recognise the rotary telephone.

  • Updated recognisable picture optotypes
  • Tumbling E and Landolt C for pre-literate patients
  • Hands-free video fixation via keyboard shortcut or USB foot pedal
  • Multiline pediatric chart view

See the Accessories page for the compatible USB foot pedal.

Options

Tailor each exam without switching applications or reconfiguring your setup.

  • Mirror mode — display behind the patient via a mirror for compact exam lanes; also supports standard front-facing display.
  • Toggle 20/XX label — show or hide the acuity notation on the patient-facing display.
  • Crowding bars — add surrounding optotypes to isolate the crowding effect.
  • Reduced contrast — multiple contrast levels for contrast sensitivity testing.
  • Red/Green overlay — for binocular balance and duochrome refraction.

Color Vision

Color vision screening is integrated directly into your digital chart workflow — no separate application or hardware required.

  • Color vision screening plates on a calibrated display
  • Seamlessly integrated — no mode-switching required
  • Monitor calibration ensures accurate color reproduction
  • Screen for the most common color vision deficiencies

Special Features

Practical features built around real exam-lane workflows.

  • Undockable / Moveable Menu — in dual-monitor setups, drag the AcuityMaster control bar to the monitor closest to you. Full menu access without looking at a display 15–20 feet away.
  • USB Foot Pedal — hands-free control of the pediatric fixation video. Keep your patient fixating while you complete refraction. See the Accessories page.
  • Pinhole Acuity — simulate pinhole testing with an on-screen overlay.
  • Mirror Mode — display the chart behind the patient via a mirror for compact exam lanes.
  • Existing hardware — compatible with Windows XP through Windows 11, including the computer already in your exam lane.
