AcuityMaster Cloud — Getting Started

AcuityMaster Cloud is browser-based — no installation, no hardware lock-in. Works on Mac and Windows in any modern browser.

System Requirements (Cloud)

ComponentRequirement
BrowserSafari 16+, Chrome 110+, Firefox 115+, or Edge 110+
Operating SystemmacOS 12+ or Windows 10/11
DisplayAny monitor; second display optional for patient chart
InternetRequired (standard broadband)
InstallationNone required

How It Works

  1. Subscribe

    Choose your seat count and subscribe at the annual rate. Pricing from $249/seat/year.

  2. Open in your browser

    Navigate to AcuityMaster Cloud in Safari, Chrome, Firefox, or Edge on any Mac or Windows device. No download.

  3. Connect your patient display

    Mirror or extend to a second monitor for the patient chart. Or run single-display mode from the exam-room computer.

  4. Start testing

    All chart types, optotypes, and modes are available immediately. Updates happen automatically.
